# 数据结构C语言实验代码

本仓库包含数据结构课程的各种C语言实现代码。

## 文件说明

### 1. binary_tree.c
二叉搜索树实现，包含以下功能：
- **创建二叉树**：通过递归方式创建二叉树
- **遍历算法**：
  - 前序遍历（递归与非递归）
  - 中序遍历（递归与非递归）
  - 后序遍历（递归与非递归）
- **查找**：在二叉搜索树中查找指定节点
- **插入**：向二叉搜索树中插入新节点
- **删除**：删除二叉搜索树中的指定节点

### 2. graph.c
图的相关操作实现

### 3. linear_list.c
线性表的C语言实现

### 4. stack_queue.c
栈和队列的C语言实现

## 使用方法

每个C文件都可以独立编译运行：

```bash
gcc -o program_name source_file.c
./program_name
```

## 编译环境

- C语言标准：C99及以上
- 编译器：GCC或Clang
